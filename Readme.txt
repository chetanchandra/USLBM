Please follow the below steps to set up the grunt for file compression in your local workplace.

Reference URL: https://gruntjs.com/getting-started 

Open the commend prompt or  terminal and follow the below steps
Step 1: Installing the CLI 
	"npm install -g grunt-cli" or “sudo install -g grunt-cli”

Step 2: Installing Grunt and gruntplugins
	"npm install" or "sudo install"

Step 3: Run the task
	"grunt watch"


Notes: don’t commite node_modules folder which is created after installing grunt.
